# Projeto Social para Idosos

Uma página web desenvolvida com HTML, CSS e JavaScript para conscientizar a população sobre a violência contra a pessoa idosa. O projeto reúne informações sobre a causa, apresenta iniciativas de apoio e orienta a população sobre como denunciar casos de violência.

## Sobre o projeto

A violência contra idosos é um problema que muitas vezes permanece invisível. Este projeto busca contribuir para a conscientização da sociedade por meio de uma plataforma informativa e acessível.

A aplicação é dividida em três seções principais:

- **Home:** apresentação da campanha e conscientização sobre a violência contra idosos.
- **Projetos:** divulgação de iniciativas e organizações que apoiam a causa.
- **Como denunciar:** informações sobre os canais oficiais para denúncia e proteção da pessoa idosa.

## Objetivos

- Promover a conscientização sobre a violência contra idosos.
- Informar sobre direitos e formas de proteção.
- Divulgar projetos e organizações de apoio.
- Incentivar a denúncia de casos de violência.

## Motivação

A tecnologia pode ser uma importante ferramenta de impacto social. Este projeto foi desenvolvido para tornar informações sobre a proteção da pessoa idosa mais acessíveis, incentivando o respeito aos seus direitos e contribuindo para a prevenção da violência.
